Инструкция по разворачиванию приложения

1)Устанавливаем Django 2.2.10, Django REST framework, python
2)Создаем базу и суперюзера, если нужно меняем в settings имя_базы/пароль
3)выполняем makemigrations, migrate

Инструкция API

1)Заходим в админку под суперюзером
2)Последовательно создаем:
- Опрос
- Вопросы для опроса (три типа вопросов)
- Варианты ответов (опционально) для вопроса
получить активные опросы можно просто зайдя на локалхост
пройти опрос можно используя /answer 
посмотреть результат можно /results/код_пользователя

шаблон json при прохождении опроса:

{
   "answers":[
      {
         "user_code":"1234",  #код пользователя
         "question":2, #id вопроса
         "choices":[0]  #выбранный вариант ответа
      },
      {
         "user_code":"1234",
         "question":3,
          "choices":[2, 3] #выбранные варианты ответа
      },
      {
         "user_code":"1234",
         "question":1,
          "answr":"aswer" #ответ текстом
      }
   ]
}
